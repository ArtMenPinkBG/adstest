Microlosofts Gaming OS

Before installation, make sure that the system suits you. Check it on virtual machines: VirtualBox (recommended and free), VmWare, Default Windows. This "custom" OS has a folder on the desktop with the necessary programs to run games ALMOST without frame drops. This OS is also a little tweaked in terms of optimization, things that almost all gamers don't use are disabled (printer, fax, services, telemetry, antivirus, and so on). You can download only from Google Drive, because it is the most convenient service for me.
